## 🚦 Como Começar

### Início Rápido (5 minutos)

1. **Instalar dependências**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Iniciar Redis**:
   ```bash
   docker-compose up -d redis
   ```

3. **Treinar modelo**:
   ```bash
   python training/train_model.py
   ```

4. **Iniciar API**:
   ```bash
   python -m app.main
   ```

5. **Testar**:
   - Acesse: http://localhost:8000/docs
   - Ou leia: [QUICKSTART.md](QUICKSTART.md)
